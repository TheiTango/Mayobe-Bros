## Getting Started

### Accessing the CMS

1. Navigate to `/admin/login` on your website
2. Enter your staff email and password
3. Click "Sign In" to access the dashboard

### First Time Setup

After logging in for the first time:

1. Go to **Settings** to configure your site
2. Set up your site title, tagline, and description
3. Upload your logo and favicon
4. Configure theme preferences
5. Add social media links

## Managing Posts

### Viewing All Posts

1. Click **Posts** in the sidebar
2. Use filters to view: All, Published, Drafts, Featured
3. Search posts by title
4. Click column headers to sort

### Creating a New Post

1. Click **Posts** → **New Post** or use the "+" button
2. Fill in the form:

#### Basic Information
- **Title**: Your post title (required)
- **Slug**: URL-friendly version (auto-generated)
- **Content**: Use the rich text editor
- **Excerpt**: Short summary for previews

#### Content Editor Features

The Microsoft Word-style editor includes:

**Text Formatting:**
- Bold, Italic, Underline, Strikethrough
- Headings (H1, H2, H3)
- Text color and background color
- Paragraph alignment (Left, Center, Right, Justify)

**Lists:**
- Bullet lists
- Numbered lists

**Media:**
- Insert images from Pexels
- Upload images from your computer
- Insert image by URL
- Insert links

**Advanced:**
- Code blocks
- Blockquotes
- Undo/Redo
- HTML view

#### SEO Settings

- **Meta Title**: Custom title for search engines (60 characters recommended)
- **Meta Description**: Search result description (160 characters recommended)
- **Meta Keywords**: Comma-separated keywords

#### Publishing Options

- **Published**: Toggle draft/published status
- **Featured**: Mark as featured post (shows on homepage)
- **Publish Date**: Schedule or backdate posts
- **Author**: Post author name
- **Category**: Select content category (required)
- **Label**: Select sub-category (optional)
- **Featured Image**: Upload or select main image

### Inserting Images

1. Click the **Image** button in the editor toolbar
2. Choose from three options:

**Pexels Library:**
- Search millions of free stock photos
- Click any image to insert
- Attribution automatically included

**Upload Image:**
- Click "Choose File"
- Select image from your computer
- Max size: 5MB
- Supported: JPG, PNG, GIF, WebP

**Image URL:**
- Paste URL of any online image
- Click "Insert Image"

### Auto-Save Feature

- Posts automatically save as drafts every 30 seconds
- Look for "Draft saved" notification
- Never lose your work!

### Editing Posts

1. Go to **Posts** list
2. Click **Edit** button on any post
3. Make changes
4. Click **Save Draft** or **Publish**

### Deleting Posts

1. Find post in Posts list
2. Click **Delete** button
3. Confirm deletion (this cannot be undone)

## Managing Pages

Static pages work similarly to posts but with some differences:

### Key Differences

- No categories or labels
- Option to "Show in Menu" (adds to site navigation)
- Option to index by search engines
- Simpler SEO options

### Creating a Page

1. Click **Pages** → **New Page**
2. Fill in title, slug, and content
3. Toggle **Published** to make live
4. Toggle **Show in Menu** to add to navigation
5. Click **Publish**

### Common Page Types

- About Us
- Contact
- Privacy Policy
- Terms of Service
- FAQ

## Categories & Labels

### Understanding the Structure

- **Categories**: Main content topics (Education, Business, Technology)
- **Labels**: Subtopics within categories (Education → Schools, Colleges)

### Managing Categories

1. Go to **Categories** in sidebar
2. To add: Click **Add Category**, fill form, save
3. To edit: Click **Edit** button, modify, save
4. To delete: Click **Delete** button

**Category Fields:**
- **Name**: Display name
- **Slug**: URL-friendly version
- **Description**: Category description
- **Display Order**: Sort order (lower = first)
- **Show in Footer**: Display in footer links

### Managing Labels

1. Go to **Labels** in sidebar
2. To add: Click **Add Label**, select category, fill form
3. Labels are grouped by category
4. Click category name to expand/collapse

**Label Fields:**
- **Category**: Parent category (required)
- **Name**: Display name
- **Slug**: URL-friendly version
- **Description**: Label description
- **Display Order**: Sort order

## Comments Moderation

### Viewing Comments

1. Click **Comments** in sidebar
2. Use tabs to filter:
   - **All**: Every comment
   - **Approved**: Published comments
   - **Pending**: Awaiting moderation
   - **Spam**: Flagged comments

### Moderating Comments

**Approve a Comment:**
1. Find pending comment
2. Click **Approve** button
3. Comment appears on site immediately

**Mark as Spam:**
1. Click **Spam** button
2. Comment hidden from site
3. Can review spam later

**Delete a Comment:**
1. Click **Delete** button
2. Confirm deletion (permanent)

**Bulk Actions:**
1. Check boxes next to comments
2. Click "Select All" for all on page
3. Choose action: Approve, Spam, or Delete
4. Click action button

## Media Library

Your central hub for all images and media files.

### Viewing Media

1. Click **Media** in sidebar
2. Toggle between **Grid View** and **List View**
3. Filter by source: All, Uploaded, Pexels
4. Search by filename

### Uploading Images

1. Click **Upload Media** button
2. Opens Image Picker modal
3. Choose file from computer
4. Image saved to library automatically

### Using Images

- Copy image URL with **Copy URL** button
- View details by clicking image
- Delete unwanted images
- See file size, dimensions, and upload date

### Image Details

Click any image to see:
- Full-size preview
- Filename and original name
- File size and dimensions
- Source (Upload or Pexels)
- Upload date
- Full URL
- Quick copy and delete actions

## Site Settings

Customize your entire website from one place.

### General Settings

- **Site Title**: Your website name
- **Tagline**: Short slogan or description
- **Site Description**: Longer description (for SEO)
- **Logo URL**: Path to logo image (e.g., `/mayobebroslogo.png`)
- **Favicon URL**: Path to favicon (e.g., `/mayobebrosfavicon.png`)

### Appearance Settings

- **Default Theme**: Choose Light, Dark, or Auto (system)
- **Primary Color**: Main accent color (hex code)
- **Font Family**: Choose from popular web fonts

### SEO Settings

- **Default Meta Title**: Fallback title for pages without custom title
- **Default Meta Description**: Fallback description
- **Google Analytics ID**: Track visitor analytics (e.g., `G-XXXXXXXXXX`)
- **Google Search Console**: Verification code

### Social Media Settings

Connect your social profiles:
- Facebook Page URL
- Twitter/X Profile URL
- Instagram Profile URL
- LinkedIn Profile URL

These appear in footer and can be used for social sharing.

### Advanced Settings

- **Custom CSS**: Add site-wide custom styles
- **Custom JavaScript**: Add tracking codes or custom scripts
- **Footer Copyright**: Copyright text in footer

### Saving Settings

1. Make changes in any tab
2. Click **Save Settings** button
3. Settings apply immediately to live site
4. Success message confirms save

## Theme Management

### User-Facing Theme Toggle

Your website visitors can choose their preferred theme:

- **Light Mode**: Eye-friendly white background
- **Dark Mode**: Pure black background (OLED-friendly)
- **System Default**: Matches device theme

Toggle is in the header (top-right corner).

### Setting Default Theme

In **Settings** → **Appearance**:
1. Choose default theme for new visitors
2. Options: Light, Dark, Auto
3. Save settings
4. Visitor preference overrides default

### Color Customization

Set your brand's primary color:
1. Go to **Settings** → **Appearance**
2. Enter hex color code (e.g., `#3B82F6`)
3. Color applies to:
   - Buttons
   - Links
   - Highlights
   - Category badges

## Tips & Best Practices

### Writing Great Posts

1. **Use Descriptive Titles**: Clear, concise, keyword-rich
2. **Write Compelling Excerpts**: Hook readers in 1-2 sentences
3. **Add Featured Images**: Posts with images get more clicks
4. **Use Headings**: Break content into scannable sections
5. **Optimize for SEO**: Fill in meta title and description
6. **Preview Before Publishing**: Check how it looks

### SEO Best Practices

1. **Meta Titles**: 50-60 characters, include main keyword
2. **Meta Descriptions**: 150-160 characters, compelling call-to-action
3. **Keywords**: Use naturally, don't stuff
4. **Alt Text**: Describe images for accessibility and SEO
5. **Internal Links**: Link to related posts
6. **Publish Regularly**: Consistent schedule builds audience

### Image Guidelines

1. **Optimize File Size**: Compress images before upload (under 500KB ideal)
2. **Use Appropriate Dimensions**: 1200x630px for featured images
3. **Choose Relevant Images**: Match content topic
4. **Credit Sources**: Pexels images include automatic attribution
5. **Use WebP Format**: Better compression, faster loading

### Organization Tips

1. **Create Clear Categories**: 5-10 main topics maximum
2. **Use Labels Wisely**: 3-5 labels per category
3. **Tag Featured Posts**: Highlight best content
4. **Regular Cleanup**: Delete old drafts and unused images
5. **Backup Content**: Export important posts regularly

### Comment Moderation

1. **Check Daily**: Respond to pending comments quickly
2. **Be Consistent**: Apply same rules to all commenters
3. **Engage Positively**: Thank good comments, ignore trolls
4. **Mark Spam Aggressively**: Protect your community
5. **Enable Notifications**: Get email alerts for new comments

### Performance Tips

1. **Limit Drafts**: Publish or delete old drafts
2. **Compress Images**: Use tools before uploading
3. **Clean Media Library**: Delete unused images monthly
4. **Use Excerpts**: Don't leave blank, hurts SEO
5. **Monitor Analytics**: Check what content performs best

## Troubleshooting

### Can't Log In

- Verify email and password are correct
- Check caps lock is off
- Clear browser cache and cookies
- Contact administrator to reset password

### Images Not Uploading

- Check file size (must be under 5MB)
- Verify file format (JPG, PNG, GIF, WebP only)
- Check internet connection
- Try different browser

### Content Not Saving

- Check internet connection
- Look for error messages
- Refresh page and try again
- Copy content to notepad before refreshing

### Posts Not Appearing on Site

- Verify post is marked **Published**
- Check category is selected
- Clear browser cache
- Wait a few minutes for updates

### Pexels Images Not Loading

- Check internet connection
- Try different search terms
- Use Upload or URL option instead
- Pexels API key may need configuration

## Appendix

### Default Login Credentials

For initial setup, use the credentials provided by your administrator.

**Important**: Change your password after first login for security.

### File Locations

- **Logo**: `/mayobebroslogo.png`
- **Favicon**: `/mayobebrosfavicon.png`
- **Uploaded Media**: `/media/` (managed automatically)

### Database Tables

The CMS uses these Supabase tables:
- `posts`: Blog posts
- `static_pages`: Static pages
- `categories`: Content categories
- `labels`: Sub-categories
- `comments`: User comments
- `reactions`: Post reactions
- `site_settings`: Configuration
- `media_library`: Uploaded images
- `reviews`: Site reviews
- `newsletter_subscribers`: Email list
- `contact_submissions`: Contact form

### Browser Compatibility

Fully supported:
- Chrome/Edge 90+
- Firefox 88+
- Safari 14+
- Mobile browsers (iOS Safari, Chrome Mobile)
